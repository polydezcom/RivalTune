# RivalTune

This application allows you to configure your SteelSeries gaming mouse on Linux using rivalcfg.

##  Prerequisites

* Python 3.6 or later
* rivalcfg library
  * Installation:
    ```bash
    sudo pip3 install rivalcfg
    ```
* Update udev rules
  * Run the following command **as root**:
    ```bash
    sudo rivalcfg --update-udev
    ```
